# Using IU Technologies
IU offers an array of IT training resources available [here](https://ittraining.iu.edu/). These resources offer training on design and media, publishing, presentations, web design,
video, IU systems, etc.

Tutorials for using other campus technologies (ex. printing) can be found [here](https://kb.iu.edu/). Some campus-wide tools can be found below:

- Canvas
  - Canvas is the enterprise instructional software used by IU. A Canvas Course includes features such as discussions, file sharing, quizzes, announcements, drop boxes, and modules.
  - [Get started with setting up a Canvas Course](https://kb.iu.edu/d/bcll)
- Microsoft 365
  - Microsoft 356 is the enterprise productivity software predominantly used at IU. It includes cloud-based versions of Microsoft Office Suite software, including the collaboration tool Teams. 365 also includes the online collaboration platform SharePoint, which is primarily used for shared file spaces.
  - [Microsoft 365 at IU](https://uits.iu.edu/initiatives/microsoft-365/index.html)
- Qualtrics
  - Qualtrics is a survey software available for use by IU students, faculty and staff.
  - [Get started with Qualtrics](https://uits.iu.edu/qualtrics/)
